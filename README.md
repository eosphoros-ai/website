# DB-GPT Website

[DB-GPT](http://dbgpt.io) 

## How to develop
```commandline

# clone current project firstly!
# install docusaurus dependencies, generate .docusaurus and node_modules folder.
npm install

# build project.
sudo npm run build 

# launch project.
npm run serve
```

The default service starts on port `3000`, visit `localhost:3000`
